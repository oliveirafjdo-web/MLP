Métricas instantâneas para anúncios do Mercado Livre

Extensão Chrome (MV3) com backend Node/Express. Mostra preço, estoque, vendidos, visitas, conversão, faturamento estimado, link clicável para a loja e ranking na categoria.

Baixar código no GitHub

Recursos

  • 📊 Métricas: preço, estoque, vendidos, visitas (30d), conversão, faturamento
  • 🏬 Link direto da loja do vendedor
  • 📈 Ranking estimado na categoria (posição/total)
  • 🔄 Atualiza ao trocar de anúncio + auto-refresh 30s
  • 🔐 OAuth seguro (login no mesmo perfil do Chrome)
  • 🎨 Paleta Redutron (azul marinho + neon ciano/verde)

Como instalar

1) Backend (Node)

cd server
cp .env.example .env
# edite .env com CLIENT_ID/SECRET/REDIRECT_URI
npm install
npm start

Publique em HTTPS (Render/Railway/VPS). O backend armazena tokens com segurança.

2) Extensão (Chrome)

  1. Abra chrome://extensions
  2. Ative “Modo do desenvolvedor”
  3. “Carregar sem compactação” → selecione extension/
  4. Abra as Opções e informe a URL do backend
  5. Visite um anúncio (MLB…) e veja o card 🎉
